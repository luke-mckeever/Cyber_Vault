#HowTo

Here's a simple cheatsheet for Markdown formatting that covers the basics:
Headers:

# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6

Emphasis:

*italic* or _italic_
**bold** or __bold__
**_both_**
~~strikethrough~~

Lists:

1. Ordered list item 1
2. Ordered list item 2
3. Ordered list item 3

- Unordered list item
* Unordered list item
+ Unordered list item

Links:

[Title](URL)

Images:

![alt text](image_url)

Blockquotes:

> Blockquote

Code:

`inline code`

code block (indented with backticks and specify language):
